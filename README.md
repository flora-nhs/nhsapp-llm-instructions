# nhsapp-llm-instructions
This file supplements the LLM instructions files for the NHS Prototyping Kit with additional information and instructions to optimise using the NHS App prototyping kit with CoPilot. 

It has been developed to be supplementary to these LLM files which were optimised for use with the NHS Prototyping Kit https://github.com/edwardhorsford/NHS-LLM-documentation

Please use issues for any questions/suggestions for how to improve them!
